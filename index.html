🚀 Tilt

Advanced Computer Automation powered by Claude Sonnet 4

🐧 🍎 🪟

🎯 One-Command Install

curl -sSL install.whytilt.com | bash

📋 Requirements

  • Docker - Will be installed automatically if missing
  • Anthropic API Key - Get one here
  • 4GB RAM - Minimum system requirement
  • Internet Connection - For Docker image download

🤖 AI-Powered

Natural language automation with Claude Sonnet 4. Describe what you want in plain English.

🖥️ Visual Desktop

Full Ubuntu desktop with VNC access. Watch your automation in real-time.

🌐 Web Interface

Beautiful Next.js frontend with real-time streaming and task management.

🔧 Tool System

Extensible automation tools: browser control, JavaScript execution, network monitoring.

📊 Analytics

Built-in monitoring for Google Analytics, Adobe Analytics, and custom tracking.

💾 Persistent Data

MongoDB storage with task history, results, and user data persistence.

🚀 Quick Start

  1. Run the install command above
  2. Enter your Anthropic API key when prompted
  3. Wait for Docker image download (~2GB)
  4. Access Tilt at http://localhost:3001

🎮 Commands

tilt # Start Tilt
tilt stop # Stop Tilt
tilt update # Update to latest
tilt help # Show help

🌐 Access URLs

Frontend: http://localhost:3001

API: http://localhost:8000

Desktop: http://localhost:6080/vnc.html

VNC: vnc://localhost:5900
