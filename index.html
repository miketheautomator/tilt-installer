🎯 One-Command Install

curl -sSL install.whytilt.com | bash

📋 System Requirements

  • Docker - Will be installed automatically if missing
  • Anthropic API Key - Get one here
  • 4GB RAM - Minimum system requirement
  • Internet Connection - For Docker image download (~2GB)

🚀 Quick Start Guide

  1. Run the installation command above in your terminal
  2. Enter your Anthropic API key when prompted
  3. Wait for Docker image download (approximately 2GB)
  4. Access Tilt at http://localhost:3001
  5. Start automating with natural language commands

🎮 Available Commands

tilt # Start Tilt automation platform
tilt stop # Stop all Tilt services
tilt update # Update to the latest version
tilt help # Show detailed help information
